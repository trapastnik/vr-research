## КОНТЕКСТ ПРОЕКТА

Разрабатывается **межведомственная инновационная экосистема художественной VR-фиксации, архивирования и дистрибуции культурных проектов Санкт-Петербурга**.

**Три столпа:** Производство → Дистрибуция → Интеграция

- **Оператор:** Библиотека им. В.В. Маяковского
- **Запуск:** 2026 (Год культуры)
- **Целевые группы:** Школьники, пожилые, маломобильные, ветераны СВО
- **Миссия:** Сохранение нематериального наследия, социальная инклюзия, патриотическое воспитание через иммерсивный опыт
- **Механизм финансирования:** 2-5% из существующих государственных субсидий на VR-отчётность
- **География фокуса:** Санкт-Петербург с перспективой масштабирования на регионы РФ

## ОБЩИЕ ТРЕБОВАНИЯ КО ВСЕМ ГЛАВАМ

### Формат:
- Язык: **русский**
- Формат: **Markdown (.md)**
- Объём каждой главы: **3 000–8 000 слов**
- Обязательные элементы: заголовок, оглавление, введение, основной текст, таблицы/схемы, выводы по главе, список источников
- Каждое фактическое утверждение сопровождается **ссылкой на источник** (автор/организация, название, год, URL если доступен)

### Данные:
- **Не старше 2023 года**, приоритет **2024-2025**
- Перекрёстная верификация: каждый ключевой факт подтверждается **2+ источниками**
- Приоритет первоисточников: официальные отчёты, пресс-релизы, научные публикации
- При отсутствии точных данных — экспертные оценки с указанием методологии

### Стиль:
- Аналитический, без рекламного языка
- Конкретные цифры, даты, имена
- Сравнительные таблицы для всех количественных данных
- Практические выводы для проекта Санкт-Петербурга в конце каждого раздела
