# 📊 Мониторинг Telegram-каналов

## Годовой анализ: март 2025 — март 2026

!!! info "Методология"
    Автоматизированный мониторинг **8 профильных Telegram-каналов** с помощью Telethon API.
    Собрано **15 482 сообщения**, из которых **3 393 (22%)** классифицированы как релевантные
    по 13 тематическим категориям, связанным с VR, AI и культурным наследием.
